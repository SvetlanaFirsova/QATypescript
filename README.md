# QATypescript

# How to install NodeJS and NPM?
Node.js provides a runtime environment for executing JavaScript outside of the browser.
https://nodejs.org/ — Install the LTS version

Node = runtime (like the Python interpreter)
npm = package manager (like pip)
package.json = requirements.txt + setup.py + scripts

# How to setup TS project
In Visual Studio Code in terminal execute:
mkdir ts-migration
cd ts-migration
npm init -y
npm install typescript --save-dev
npx tsc --init
